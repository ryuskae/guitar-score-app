빈 마디를 클릭한 뒤 키보드 A–G로 음표를 입력하세요. TAB 숫자를 클릭하면 바로 편집할 수 있습니다.

키보드 입력

A–G(한글 입력 상태에서도 물리 키 기준): 기본 한 옥타브 낮은 음 입력 · 1–5: 온음표, 2분음표, 4분음표, 8분음표, 16분음표 선택 · =: 샵 · -: 플랫 · TAB 숫자 선택 중 숫자: 프렛 변경 · . : 점음표 켜기/끄기 · ←/→: 음표 이동 · 오선 선택 후 ↑/↓: 반음 이동 · TAB 선택 후 ↑/↓: 같은 음을 인접 줄로 이동 · Ctrl+↑/↓: 한 옥타브 이동 · Tab: 선택 마디 뒤에 새 마디 삽입 · Enter: 선택 마디 다음에서 줄바꿈 · V: 이미 추가된 성부 전환 · Delete: 같은 길이의 쉼표로 변경 · 음표를 드래그: 범위 선택 후 바레

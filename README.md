# Guitar Score Editor

브라우저에서 오선 악보와 기타 TAB을 함께 편집하는 사보 앱입니다. 별도 설치 없이 GitHub Pages에서 실행됩니다.

## 구현된 기능

- 오선 음표와 TAB 숫자의 양방향 연동
- TAB 숫자 직접 선택, 숫자 키 프렛 변경, 화살표 줄/음표 이동
- 점음표, 쉼표, 꾸밈음, 타이, 오선·TAB 슬러, 클래식 기타 바레 범위
- 4개 성부와 성부별 색상 표시
- Enter 키 마디 줄바꿈, A4/A3/Letter/화면 페이지 크기
- 제목·작사·작곡 정보, MusicXML 불러오기/내보내기, 인쇄 기반 PDF 저장
- 클래식 기타·통기타·일렉 기타·피아노 문서 선택
- 한글 IME 상태에서도 물리 키 A–G 음표 입력

`index.html`을 열거나 정적 웹 서버로 저장소 루트를 제공하면 실행할 수 있습니다.
